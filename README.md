# Conrad-Auralis-One
Auralis One is an AI-powered mental health app that uses Vision Transformer (ViT) models for facial emotion recognition to detect early signs of anxiety. Built with Python, PyTorch, and React Native, it provides real-time emotional insights, trend tracking, and chatbot support. The app is currently being developed using Xcode in Swift.
